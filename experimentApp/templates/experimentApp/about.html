About

This application was developed to create a simpler, more customised application to create and test experiments on (compared to existing websites such as Google Forms). Experiments comparing images and videos can be run here, more variations may be added in the future. Functionality includes:

  • Creating experiments, with the ability to modify the experiment through customisable options
  • Letting users take such experiments
  • Exporting results as .csv
  • Basic account system to prevent unregistered users from creating/modifying experiments

Registered users can create experiments for a given set of images or videos. These files must include a "base" set of files, and another set of files for algorithms applied to the base set. There currently exist three types of experiments to compare files:

  • Pairwise Comparison - Choose which files is best
  • Ranking - Organise files set in order of best to worst
  • Rating - Assign value to each files in set

Results are stored in a PostgreSQL database, admins can export them as a .csv file.
