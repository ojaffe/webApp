About

This application was developed to create a simpler, more customised application to create and test experiments on (compared to existing websites such as Google Forms). Experiments comparing images can be run here, more variations may be added in the future. Functionality includes:

  • Creating experiments, with the ability to modify the experiment through customisable options
  • Letting users take such experiments
  • Viewing the results of such experiments and exporting as .csv
  • Basic account system to prevent unregistered users from creating/modifying experiments

Registered users can create experiments for a given set of images. These images must include a "base" set of images, and another set of images for algorithms applied to the base set. There currently exist types of experiments to compare images:

  • Pairwise Comparison - Choose which image is best
  • Ranking - Organise image set in order of best to worst
  • Rating - Assign value to each image in set

Results are stored in a [FOR NOW] SQLite database, admins can view such results in browser or choose to export them as a .csv file.
